VSCode 단축어 메모

파일 관리

기능 단축어
파일 열기 Ctrl + O
파일 저장 Ctrl + S
모든 파일 저장 Ctrl + K, S
파일 닫기 Ctrl + W
새 파일 만들기 Ctrl + N

편집

기능 단축어
찾기 Ctrl + F
바꾸기 Ctrl + H
주석 처리 Ctrl + /
코드 포맷 Shift + Alt + F

탐색

기능 단축어
파일 탐색기 열기/닫기 Ctrl + B
사이드바 열기/닫기 Ctrl + Shift + E

명령 및 터미널

기능 단축어
명령 팔레트 열기 Ctrl + Shift + P
터미널 열기/닫기 Ctrl + `

디버깅

기능 단축어
디버그 시작/중지 F5 / Shift + F5
디버그 다음 단계 F10
디버그 진입 F11
디버그 나가기 Shift + F11
