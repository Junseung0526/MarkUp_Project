Git 명령어 시트

명령어 설명
git init 새 Git 저장소 초기화
git clone <repository> 기존 저장소 복제
git status 작업 디렉토리 상태 표시
git add <file> 파일을 스테이징 영역에 추가
git commit -m "메시지" 메시지와 함께 변경 사항 커밋
git push 원격 저장소에 변경 사항 푸시
git pull 원격 저장소에서 변경 사항 가져와 병합
git branch 브랜치 목록, 생성 또는 삭제
git checkout <branch> 다른 브랜치로 전환
git merge <branch> 현재 브랜치에 다른 브랜치 병합
git log 커밋 기록 표시
git remote -v 원격 저장소 표시
git fetch 다른 저장소에서 객체와 참조 다운로드
git reset <file> 파일 스테이징 취소
git rm <file> 작업 디렉토리와 스테이징 영역에서 파일 제거
git revert <commit> 지정된 커밋의 변경 사항 되돌리기
git reset --hard <commit> 지정된 커밋으로 되돌리고 이후의 변경 사항 삭제
git restore <file> 작업 디렉토리에서 파일 복원
